# Linux Start
The simplest way of installing everything you need in a Debian-based Linux system.

## Instructions
```
$ sudo apt install git
$ git clone https://github.com/holocene-epoch/linux-start
$ cd linux-start
$ ./start.sh
```

## Packages installed
- vim 
- gnome-core 
- gnome-tweaks 
- vlc 
- steam 
- lutris 
- wine 
- chromium 
- discord
